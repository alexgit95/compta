🔗 Export via API

L'export est également accessible via l'endpoint REST suivant, protégé par clé API.

GET
Header : Authorization: ApiKey <votre-clé>
ou : X-Api-Key: <votre-clé>
Gérez vos clés API dans l'onglet 🔑 Clés API.
🔐 Maintenance WebAuthn / Clés de passage

En cas de problème avec les clés de passage (erreur lors de l'enregistrement), vous pouvez effacer toutes les données WebAuthn. Les utilisateurs devront réenregistrer leurs clés.
